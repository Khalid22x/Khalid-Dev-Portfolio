My Graduation Project: Building a Real-Time Object Detection System

By Khalid Abd Elhamid Abd El Hafeez | Published on: May , 2024

Welcome! For my graduation project, I dove into the fascinating field of Artificial Intelligence to build a real-time object detection system. Computer vision is a critical branch of technology that allows machines to process images and identify objects within them, much like humans do. My project focused on creating a system capable of detecting objects like people and cars in digital images and videos.

"Object detection has widespread applications across various fields of computer vision, making it a prominent area of research in artificial intelligence."

The Motivation: Why This Project?

My primary motivation was an intense passion for learning artificial intelligence techniques. While researching the topic, I noticed several gaps in existing applications: many had a poor Graphical User Interface (GUI), some were slow and lacked accuracy, and most paid applications were not available for mobile. This inspired me to create a system that was not only functional but also efficient and user-friendly.

My Objectives and Scope

The main goal was to create a real-time object discovery system that is compact and efficient enough to run on various devices. To achieve this, the project scope included several key aims:

  • Implementing a small and fast object detection model.
  • Creating an object dataset and using it to train the model.
  • Validating and analyzing techniques to improve both the accuracy and speed of the detection model.

The Technology Stack

Building this system required a combination of powerful libraries for both the AI model and the user-facing website.

For the Model:

  • OpenCV-Python
  • Tensorflow
  • Numpy
  • Pillow
  • Matplotlib
  • Easydict

For the Website:

  • Flask
  • Jinja2
  • Gunicorn
  • Werkzeug
  • Click

The Process: From Concept to Classifier

The project followed a structured plan. The core workflow involved loading the object detection data, performing image preprocessing, and then using that data to generate and train the object detector classifier. Once trained, the classifier was serialized and saved so it could be loaded and applied to new test data from images, videos, or a live camera stream. The system was designed to allow a user to load an image or open a camera to detect objects in real-time.

Below is a screenshot of the web application successfully identifying objects in an uploaded image.

Putting the Model to the Test

I conducted several tests to validate the model's effectiveness under different circumstances. The model performed effectively when detecting objects in standard pictures, videos, and even in real-time via a camera feed. It also succeeded in detecting objects in black and white photos.

However, one key learning experience came from a test case that was marked "Ineffective": the model struggled with detecting objects in pictures taken from a great distance. This highlighted a clear area for future improvement and a deeper understanding of the model's limitations.

Future Plans and Enhancements

This project serves as a strong foundation, and I have several ideas for future work to make it even more robust and versatile. My plans include:

  • Developing a mobile app version of the application.
  • Connecting the system to specialized hardware for dedicated performance.
  • Expanding the model into a complete self-sufficiency system that includes face and traffic light recognition.
  • Adding a night vision mode to enable visual tracking in low-light conditions.

Conclusion & Key Takeaways

Object detection is a fundamental capability for many modern applications, from consumer electronics to advanced driving assistance systems. While the field has made incredible progress, there is still work to be done to reach human-level performance, especially in adapting to new environments. This project was a rewarding journey that solidified my skills and left me excited to continue exploring the vast potential of computer vision.
